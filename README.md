## React Youtube

### Installation

- Clone this respository
- Goto the the directory in which application has been cloned
- Run npm command to install dependencies

```sh
npm install
```

### Run on development environment
```sh
npm run open:src
```

### Run for production environemnt
```sh
npm run build
npm run start
```
